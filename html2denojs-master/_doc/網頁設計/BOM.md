# 瀏覽器物件模型 BOM 


JavaScript 是為了瀏覽器而設計的，因此瀏覽器提供了一組物件給 JavaScript 操控，這組物件稱為 Browser Object。 (其整體形成一個 Model ，我們稱之為 Browser Object Model, BOM)

以下是常見的瀏覽器物件：

* window
    * window.history
    * window.location
    * window.document
    * window.screen
