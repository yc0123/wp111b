# Visual Studio Code

## 安裝

* https://code.visualstudio.com/

## 使用

* 建議開啟資料夾，而非單獨檔案
* 在檔案上按滑鼠右鍵選 open in terminal 可以開啟終端機 (或者用 View/Terminal)
* 有很多插件可以安裝 (選 extension)

## 快捷鍵

* Ctrl-K-C : 整塊成為註解
* Ctrl-K-U : 解除註解
