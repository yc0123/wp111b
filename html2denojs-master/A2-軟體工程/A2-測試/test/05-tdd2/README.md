## 練習 -- TDD2

講解： 從 lodash 中再度挑選第二個函數，但是這次採用 TDD 的方式。

TDD : Test Driven Development  (先寫測試，再寫程式)

1. 挑選第二個 lodash 函數。
2. 選定後寫測試函數，寫好後直接測試，此時會報錯誤。
3. (TDD2) 寫完後再測試，就會回報測試結果 (若成功就沒有錯誤)
