3.1 -- PyJaDown 編程環境

類似 Jupiter ，但支援 python + javascript + markdown，並用《表格+物件》溝通。用 markdown 寫文件，用 python 做科學計算，用 javascript 做前端網站與資料互動呈現。

  • 類似產品 -- https://github.com/fastai/nbdev

使用 eval 執行程式片段

  • 類似產品 -- https://www.programiz.com/python-programming/methods/built-in/eval

  const Ui = {}

  Ui.id = function(path) {
    return document.getElementById(path)
  }
  
  Ui.one = function(path) {
    return document.querySelector(path)
  }
